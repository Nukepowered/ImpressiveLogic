# ImpressiveLogic
ImpressiveLogic is Minecraft mod to allow you build complex binary circuits.

### Building / Dev setup

Setup of environment can be found [here](https://mcforge.readthedocs.io/en/1.18.x/gettingstarted/#from-zero-to-modding)

To build

```
$ gradlew build
```
